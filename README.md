# Sparse Matrix Manager

## Description
This project is a C program that manages sparse matrices using a singly linked list.  
It allows efficient handling of matrices with many zero elements.

## Features
- Convert a dense matrix to a sparse linked list
- Insert new non-zero elements at beginning, end, or random positions
- Delete elements by position
- Search elements by row and column
- Reconstruct the dense matrix from the linked list
- Find the largest non-zero value
- Identify the row and column with the most non-zero elements
- Sum two sparse matrices
- Compare memory usage between dense array and linked list
- Save and load sparse matrix from a text file

## Concepts Used
- Pointers
- Linked Lists
- Dynamic Memory Allocation
- File Handling
- Random Number Generation
